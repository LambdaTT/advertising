# lambdatt-php/advertising

A SplitPHP Framework plugin containing a advertising control system.

---

## Installation

Install via Composer:

```bash
composer require lambdatt-php/advertising
```

Run the Migrations:
```bash
php console migrations:apply --module=advertising
```

**PS: this can only be installed on a SplitPHP Framework project. For more information refer to: https://github.com/splitphp/core**
